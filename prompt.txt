Data source:

Use this JSON endpoint:

https://raw.githubusercontent.com/nukIeer/gameshieldcdn/main/games.json

        "downloads": "100M+",
        "rating": "4.56",
        "size": "39.5MB",
        "ageRating": "PEGI-12",
        "androidVersion": "8.1.0+",
        "version": "1.0.0"
      },
      "description": "Fast-paced multiplayer action game.",
      "whatsNew": "Bug fixes and improvements.",
      "media": {
        "iconUrl": "https://raw.githubusercontent.com/nukIeer/gameshieldcdn/main/games/brawlstars/icon.png",
        "bannerUrl": "https://raw.githubusercontent.com/nukIeer/gameshieldcdn/main/games/brawlstars/banner.png",
        "screenshots": [
          "https://raw.githubusercontent.com/nukIeer/gameshieldcdn/main/games/brawlstars/screenshots/1.jpg"
        ]
      },
      "downloadLinks": {
        "playStoreUrl": "https://play.google.com/store/apps/details?id=com.supercell.brawlstars",

        "apk1": null,
        "apk2": null,
        "mirrors": [
          "https://dummy-mirror1.com"
        ]
      }
    }
  ]
}

Download button logic:
Implement simple MVP button behavior.

Button state:
- If apk1 exists → button says "Download"
- Else if apk2 exists → button says "Download"
- Else if mirrors list has at least one item → button says "Mirror"
- Else if playStoreUrl exists → button says "Store"
- Else → button disabled, says "Unavailable"

On button tap:
- If apk1 exists → open apk1 URL
- Else if apk2 exists → open apk2 URL

- Else if playStoreUrl exists → open playStoreUrl
